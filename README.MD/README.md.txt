# 🛍️ Product Catalog Manager - iBee Analytics Assessment

This is a simple web application for managing a product catalog. Built with **FastAPI** (backend) and **HTML/JavaScript** (frontend).

---

## 🚀 Features

- View API status
- Add products
- View product list (stored in memory)

---

## ⚙️ Tech Stack

- Python (FastAPI)
- HTML, CSS, JavaScript (Vanilla)
- No database (in-memory storage)
